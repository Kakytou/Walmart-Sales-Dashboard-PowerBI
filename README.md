# Walmart Sales Dashboard - Power BI

## Sobre o Projeto

Este projeto foi desenvolvido utilizando Power BI com o objetivo de analisar o desempenho de vendas das lojas Walmart e identificar fatores que impactam os resultados.

## Objetivos

* Analisar a evolução das vendas ao longo do tempo.
* Identificar as lojas com melhor desempenho.
* Avaliar o impacto dos feriados nas vendas.
* Explorar indicadores relevantes para tomada de decisão.

## Ferramentas Utilizadas

* Power BI
* Power Query
* DAX

## Principais Indicadores

* Total de Vendas
* Média de Vendas
* Melhor Mês em Vendas
* Loja Líder em Vendas

## Principais Insights

* Semanas com feriados apresentaram vendas médias superiores às semanas comuns.
* A Loja 4 apresentou o maior volume acumulado de vendas.
* Foram identificados picos sazonais de vendas ao longo dos anos analisados.
